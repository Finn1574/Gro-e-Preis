Der Große Preis

Willkommen! Dieses Projekt besteht aus einem React-Frontend und einem Node.js-Backend. Die GitHub Pages Vorschau zeigt lediglich diese statische Übersicht. Um das vollständige Quiz mit Host- und Player-Flows zu nutzen, starte die Anwendung lokal.

Zur Dokumentation Repository ansehen

Lokale Entwicklung: npm install, anschließend in getrennten Terminals npm run dev:server und npm run dev:web.

Die React-App findest du im Ordner /web, den Express-Server im Ordner /server.
